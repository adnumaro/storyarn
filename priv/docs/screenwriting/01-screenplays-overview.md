The screenplay editor lets you write scripts in **industry-standard format** — action lines, character cues, dialogue, parentheticals, and transitions. It's a full writing environment designed for game narrative.

## When to use Screenplays

- **Standalone scripts** — cutscenes, cinematics, linear sequences
- **Flow sync** — generate formatted scripts from your flow dialogue
- **Voiceover scripts** — formatted for recording sessions
- **Design documents** — narrative pitch documents in professional format

## Elements

Screenplay content is built from standard elements:

| Element | Purpose | Example |
|---------|---------|---------|
| **Scene Heading** | Location and time | INT. TAVERN - NIGHT |
| **Action** | Stage directions and descriptions | *The door creaks open.* |
| **Character** | Who's speaking | JAIME |
| **Dialogue** | What they say | I've been waiting for you. |
| **Parenthetical** | How they say it | (whispering) |
| **Transition** | Scene transitions | CUT TO: |

## Organizing screenplays

Like sheets and flows, screenplays support a **tree structure** with folders:

```
Act 1/
├── Opening Cinematic
├── Tavern Introduction
└── First Quest
Act 2/
├── The Betrayal
└── Forest Chase
```

## Fountain export

Export any screenplay to **Fountain format** (`.fountain`) — the plain-text screenplay format supported by most professional screenwriting tools:

- Final Draft
- Highland
- WriterSolo
- Fade In

## Flow synchronization

Link a screenplay to a flow to **automatically sync dialogue**. When flow dialogue changes, the screenplay reflects those changes. This keeps your scripts and interactive dialogue in sync without manual updates.
